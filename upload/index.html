Formulir Upload File

Silakan isi formulir di bawah ini untuk mengunggah file Anda. Pastikan file yang diupload sesuai dengan ketentuan yang berlaku.

Ketentuan Upload:

  • File dalam format .pdf, .docx, .xlsx, atau .jpg
  • Ukuran maksimal file: 10MB
  • Nama file tidak boleh mengandung karakter khusus

Jika formulir tidak muncul, silakan klik di sini untuk membuka formulir di tab baru.
